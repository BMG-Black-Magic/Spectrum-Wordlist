# Spectrum-Wordlist
A comprehensive wordlist built on adjectives.txt nouns.txt and numbers.txt using a simple python script, Final wordlist is ~200GB 


# Instructions
clone using `git clone https://github.com/BMG-Black-Magic/Spectrum-Wordlist`
`python wordlist.py`
it is a 200GB file so it *will* take quite some time to build depending on your CPU.
